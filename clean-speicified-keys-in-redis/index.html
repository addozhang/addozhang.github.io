Redis清理缓存

2016-12-13
笔记

文章目录

最近有个需求需要主动的去清理部分缓存，考虑的原子性的问题，用Lua脚本进行实现。

Lua脚本

1
2
3
4
5
6
7

local count = 0
for _,k in ipairs(redis.call('KEYS', ARGV[1]))
    do
       redis.call('DEL', k)
        count = count + 1
    end
return count
